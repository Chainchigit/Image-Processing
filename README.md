# Dual USB Camera Capture with Python

This project captures images from **two USB cameras simultaneously** using Python and saves them to a specified folder.  
It also supports saving directly into a **Google Drive Desktop sync folder** for automatic cloud backup.

---

## Features
- Open **two USB cameras at the same time**
- Capture images from both cameras simultaneously
- Automatic capture every configurable interval (default: 60 seconds)
- Save images to a local or Google Drive synced folder
- Simple keyboard controls for manual capture and exit

---

## Tech Stack
- **Python 3.8+**
- **OpenCV (opencv-python)**
- Windows 10/11 environment
- Optional: **Google Drive for Desktop** for auto-sync
